# sublime text 3 最新激活码

> 在使用Sublime时会定期弹出购买提示框，避免出现购买提示，影响工作效率，我们可 以使用网上的激活码，虽然有些不厚道，但是工作以后，一定选择购买正版支持一下。
>
> 打开sublime菜单 > help > enter license,输入以下3个激活码其中一个即可。
> 2016年2月9日 sublime text3更新到3103版本，之前的激活码无法使用了，所以激活码也得更新。
>
